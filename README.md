# nf-ampliseq-test
Repo for quick testing of nf-ampliseq

------------------------------------------------------------------------

#### Test

To test nf-ampliseq execution using test data, run:

    bash runtest.sh

Your console should print the Nextflow log for the run, once every process has been submitted, the following message will appear:

     ======
     
      nf-ampliseq TEST SUCCESSFUL
      
     ======

nf-ampliseq results for test data should be in the following path:

  ./results
